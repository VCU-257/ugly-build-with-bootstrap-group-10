## 1. My Assigned Work
I chose to work on the New Bucket screen, where users input information for a new saving project. Despite it not being officially assigned to me, I also decided to create the Dashboard screen so that the user can fully navigate through flow B. ## 2. Bootstrap Implementation
For the New Bucket screen, I had to use a lot of form-control/form-labels to build all the options.  I also had to use many containers, some buttons, and I used row-gap to manage gaps between form elements, and col-## to control horizontal scaling.
